Sobre o Dashboard

Este dashboard interativo apresenta dados de Hemovigilância em Pós-Mercado fornecidos pela ANVISA (Agência Nacional de Vigilância Sanitária). Os dados incluem notificações de eventos adversos relacionados ao ciclo do sangue e hemoderivados.

Funcionalidades:

  • Visão Geral: Métricas principais e tendências temporais
  • Distribuições: Análises por região, tipo de evento e período
  • Mapa Brasil: Visualização geográfica interativa das notificações
  • Correlação: Análises comparativas entre variáveis
  • Dados: Acesso direto às tabelas e filtros detalhados
Como Usar os Filtros

Use o painel de filtros no menu lateral para refinar os dados:

  1. Selecione os Estados (UF) desejados
  2. Escolha o Tipo de Evento a analisar
  3. Defina o período com os Anos
  4. Clique em Aplicar Filtros

Os gráficos e tabelas serão atualizados automaticamente com os dados filtrados.

Exportar Dados

Você pode exportar os dados filtrados em formato CSV:

  1. Aplique os filtros desejados
  2. Acesse a página Dados
  3. Clique no botão Baixar CSV

O arquivo será salvo com os dados filtrados em seu computador.
